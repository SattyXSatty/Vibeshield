# 🔑 Configure Your API Key

VibeShield uses an advanced reasoning engine called **Cortex-R**, powered by Google's Gemini models, to analyze your code intent and autonomously execute UI tests.

To use VibeShield, you must provide a valid API key.

### How to set it up:
1. Get a free API key from [Google AI Studio](https://aistudio.google.com/apikey).
2. Open your VS Code Settings (`Cmd+,` or `Ctrl+,`).
3. Search for `vibeshield.apiKey`.
4. Paste your key into the text box.

> **💡 Note:** You can also configure this directly inside the VibeShield Overlay UI by clicking the gear ⚙️ icon in the top right.
